This app is a simple contact manager, using angular and bootstrap for the front end, express and node for the backend, and mongo db for the database!
Its using MongoDB! so you have to configure the route of the DB in routes/api.js file.
(Mongo creates the database and the colleccion if its doesn't exist.)
Database: agenda
Collection: contacts

This app runs on a node js server,so, just open the command line, go to the main directory  and run "node app.js".
